## Lab3 基于UDP的可靠传输:
   1. 建立连接、差错检测、确认重传等，流量控制采用停等机制，完成给定测试文件的传输
   2. 基于滑动窗口的流量控制机制，采用固定窗口大小，支持累积确认，完成给定测试文件的传输
   3. 选择实现一种拥塞控制算法，也可以是改进的算法，完成给定测试文件的传输
   4. 基于给定的实验测试环境，通过改变延迟时间和丢包率，完成下面3组性能对比实验：
        * 停等机制与滑动窗口机制性能对比；
        * 滑动窗口机制中不同窗口大小对性能的影响；
        * 有拥塞控制和无拥塞控制的性能比较。
